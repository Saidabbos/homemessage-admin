# 🏠 Sabai - Admin Panel Login Guide

## 🎯 Quick Start

### 1. Open Admin Login Page
```
URL: http://localhost/admin/login
```

### 2. Admin Credentials
```
Email:    admin@example.com
Password: password
```

### 3. Click Login

### 4. View Dashboard
```
You'll see admin dashboard with statistics and navigation menu
```
